Программа для вычисления маршрута с наименьшими денежными затратами
Автор
Сандросян Давид

Программа позволяет вычислить самый дешевый маршрут из первого города в последний город. На преодоление одной дороги требуется один бак бензина, цена за который задается в городе прибытия.

Формат входных данных

На вход программы подается файл в формате .csv, содержащий матрицу смежности

Cодержание выходного файла

Выходной файл представляется в виде одного текстового файла, содержащего результат вычисления.
В случае ошибки во входных данных, сообщение об ошибке выводится в консоль, выходной файл при этом не создается
